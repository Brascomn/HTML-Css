Pseudo-Classes

Uma pseudo-classe CSS é uma palavra-chave adicionada a seletores que especifica um estado especial do elemento selecionado Seletores Personalizados. São as pseudo-classes, tem que estar relacionadasa um elemento ou classe, sabendo dos estado e podendo configura-lás.

Os Pseudo-Classes são marcadas com dois-pontos ( : ).

    Abaixo, vão alguns deles:

    :hover – quando passamos o mouse em cima do elemento.
    Passe Aqui o Mouse


    MouseOn


    :active – quando ativamos o elemento. :visited – quando o link é visitado.
    :focus – quando um elemento recebe foco. Muito utilizado em campos de texto. 

            
                . = class
                : = pseudo-class
                :: = pseudo-element
                > = children

Pseudo-Elementos

    Um pseudo-elemento CSS é uma palavra-chave adicionada a um seletor que permite que você estilize uma parte específica do elemento selecionado.
    Os Pseudo-Elementos são marcadas com quatro-pontos ( :: ).

      Abaixo, vão alguns deles:

      ::After-Adição de configurações.

      ::before -Antecipação de configurações.

      ::After

      ::before
